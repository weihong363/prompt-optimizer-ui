# Prompt Interpreter UI

这是一个功能强大的提示词解释器前端应用，支持语音输入、实时转录和智能提示词分析。

## 主要功能

- 支持语音输入和实时转录
- 可视化波形音频反馈
- 智能提示词处理和分析
- 响应式设计，适配各种设备
- 多语言支持（默认中文）

## 技术栈

- React 18
- Vite
- JavaScript
- Web Speech API
- CSS3 动画和响应式设计

## 代码优化内容

我们对项目进行了全面的代码优化，以提高性能、可维护性和用户体验：

### 1. 组件优化
- 优化了App.jsx组件，简化了组件结构和状态管理
- 提取了SpeechIndicator和Modal为独立子组件，增强了代码模块化
- 改进了条件渲染逻辑，减少不必要的渲染

### 2. 语音输入优化
- 重构了SpeechInputButton组件，优化了音频分析逻辑
- 添加了统一的资源清理函数，避免内存泄漏
- 优化了波形可视化渲染，使用Array.map替代循环
- 改进了音频分析器参数配置，提高了响应速度

### 3. CSS样式优化
- 扩展了全局重置样式，确保跨浏览器一致性
- 整合了所有动画定义到统一区域，便于管理
- 优化了焦点样式，使用:focus-visible提升可访问性
- 合并了重复的CSS样式定义，减少代码冗余
- 添加了样式分类注释，提高代码可读性

### 4. 工具函数优化
- 增强了API调用函数，添加了超时处理和错误分类
- 移除了重复的SpeechRecognitionService实现
- 优化了useSpeechRecognition Hook，添加了更健壮的错误处理
- 增加了资源自动清理机制，避免内存泄漏

## 开发指南

### 安装依赖
```bash

```

### 启动开发服务器
```bash
npm run dev
```

### 构建项目
```bash
npm run build
```

```bash
npm run preview
```
